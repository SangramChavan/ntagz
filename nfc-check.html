Frequently Asked Questions

How do I know if my smartphone has NFC built-in? +
You can use an online hardware check tool like the ntagz NFC Detector, or verify manually inside your OS. On Android devices, open Settings > Connected Devices > Connection Preferences to look for an NFC toggle switch. All Apple iPhones starting from iPhone 6 and up feature built-in contactless hardware modules.
Why does my browser say Web NFC is not supported? +
Web NFC API structures rely on secure configuration handshakes. Apple blocks iOS web layouts (Safari, Chrome, Firefox) from querying hardware antennas directly due to proprietary ecosystem guidelines. To safely use web scanning tools, access the module over an updated build of Google Chrome running on an Android platform.
Can I read and write NTAG chip units directly from a website? +
Yes, systems supporting native Web NFC capabilities (like Android Google Chrome) can interactively read, update, and program NTAG213, NTAG215, and NTAG216 coin and card components natively inside your browser without demanding standalone third-party app installations.
